# Library Documentation

This project uses a small, curated set of libraries.

Only introduce a new library if it provides significant value.

---

# Frontend

## React

Purpose

Build the user interface.

Rules

- Functional components only.
- Use hooks.
- Reusable components.

## Tailwind CSS

Purpose

Styling.

Rules

- Use utility classes.
- Avoid inline styles.
- Follow ui-tokens.md.

---

## Lucide React

Purpose

Icons.

Rules

- Use consistent icon sizes.
- Prefer outline icons.

---

## React Flow

Purpose

Knowledge Map.

Rules

- Only use for concept visualization.
- Keep layout clean.
- Animate mastery updates.

---

## Framer Motion

Purpose

Animations.

Rules

- Subtle only.
